LUMINATOR TECHNOLOGY GROUP (May, 2017 - Present)
Luminator is a company that produces technology for mass transit (bus, rail, and air).
Their technology includes standalone units as well as systems that integrate embedded
software and web applications. The workflow is moving towards being test-driven (Agile).

My primary role as an intern has been software testing, and I have worked with nearly
all of the products the company currently produces. As a result, I have been exposed to
each of the different areas of software engineering in this company. Below, I highlight
some of the most valuable experiences I gained while working for this company.

AUTOMATION
I began exploring automated testing using Selenium Webdriver and Java by automating the
basic functionality of an existing web application. After learning the basics, I created
an entire testing environment in C# using .NET framework and implemented the Page Objects
model. My appreciation for object oriented programming grew immensely as I developed the
testing environment.

EMBEDDED SYSTEMS
My exposure to embedded systems has been quite thorough. I work closely with developers to
to test proprietary technology. I also spend a considerable amount of time working to test
the integration of that technology with existing and new hardware. I have begun familiarizing
myself with basic Linux CLI commands and have become comfortable with the operating system.
